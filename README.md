# Kashish Singh Portfolio

A clean, minimal portfolio website built with HTML, CSS, and JavaScript.

## Features
- **Responsive Design**: Works perfectly across all device sizes.
- **Interactive Global Footer**: Featuring a sitemap and contact icons.
- **Operational Standards**: Highlighting core design principles with interactive cards.
- **Testimonials Carousel**: Displaying client and collaborator feedback.
- **Case Studies**: Deep dives into design projects and human-centric workflows.

## Technology Stack
- HTML5
- Vanilla CSS
- JavaScript
